# Word Randomizer — имитация рукописного текста в Word
===

# 

# Приложение рандомизирует текст `.docx`:

# \- размер шрифта ±1 pt  

# \- случайные пробелы  

# \- дрожание символов  

# \- микросмещения baseline  

# \- эффект «волны руки» (плавное дрейфование текста)  

# \- неровные интервалы между строками  

# \- лёгкая порча kerning и символов  

# 

# \## Запуск

# 

# 1\. Скачать `randomize\_word\_text.exe`

# 2\. Открыть и выбрать `.docx` файл

# 3\. Получить результат с `\_random\_font.docx`

#
